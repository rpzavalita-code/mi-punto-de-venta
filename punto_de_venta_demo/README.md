# Punto de Venta Web/Móvil

Demo de sistema de inventario con React. Despliegue rápido en Vercel o Netlify.

## Instrucciones Rápidas
1. `npm install`
2. `npm run dev` para desarrollo local
3. Subir a GitHub y desplegar en Vercel

Incluye:
- Registro de productos por código
- Inventario editable
- Exportación a Excel
